# App Store 发布检查清单

## 开发准备
- [ ] Apple Developer Program 账号已激活
- [ ] Xcode 已安装最新版本
- [ ] Flutter 环境配置正确

## 应用配置
- [ ] Bundle Identifier 已设置且唯一
- [ ] 应用版本号已更新 (pubspec.yaml)
- [ ] 应用图标已配置 (1024x1024 PNG)
- [ ] 启动画面已配置
- [ ] 权限描述已添加 (Info.plist)

## 代码质量
- [ ] 所有功能正常工作
- [ ] 在真实设备上测试通过
- [ ] Release 模式构建成功
- [ ] 无崩溃和严重 bug
- [ ] 性能表现良好

## App Store Connect
- [ ] 应用记录已创建
- [ ] 应用元数据已填写完整
- [ ] 截图已上传 (所有必需尺寸)
- [ ] 应用描述已撰写
- [ ] 关键词已设置
- [ ] 分级已选择
- [ ] 定价已设置

## 法律合规
- [ ] 隐私政策已准备 (如需要)
- [ ] 使用条款已准备 (如需要)
- [ ] 内容符合 App Store 审核指南
- [ ] 无版权侵权内容

## 技术要求
- [ ] 支持最新 iOS 版本
- [ ] 64 位架构支持
- [ ] 无私有 API 使用
- [ ] 网络安全要求 (HTTPS)
- [ ] 无调试代码残留

## 提交前最终检查
- [ ] 构建版本已上传到 App Store Connect
- [ ] 版本信息已关联构建
- [ ] 审核信息已填写
- [ ] 联系信息已更新
- [ ] 发布选项已设置

## 审核后
- [ ] 监控审核状态
- [ ] 准备回应审核反馈
- [ ] 发布后监控用户反馈
- [ ] 准备后续版本更新

---

**注意事项：**
1. 首次提交通常需要更长审核时间
2. 确保应用功能完整，避免审核被拒
3. 保持与苹果审核团队的沟通
4. 准备应对可能的审核问题
